# Claude Code YouTube 영상 수집 및 리뷰 사이트 PRD

## 프로젝트 개요
클로드 코드(Claude Code)와 관련된 YouTube 영상을 매일 자동으로 수집하고, 사용자가 요약된 내용과 썸네일을 확인한 후 영상을 시청할 수 있는 웹사이트 구축

## 핵심 기능

### 1. 자동 영상 수집 시스템
- YouTube Data API v3를 사용한 일일 영상 수집 (cron job)
- 검색 키워드: "Claude Code", "Anthropic Claude Code", "Claude CLI" 등
- 중복 영상 제거 및 관련성 필터링
- 신규 영상 발견 시 알림 시스템

### 2. 영상 정보 관리
- 영상 메타데이터 저장 (제목, 설명, 썸네일, 채널명, 조회수, 업로드 날짜)
- AI를 활용한 영상 내용 요약 생성 (Claude API 사용)
- 태그 및 카테고리 자동 분류
- 영상 품질 및 관련성 점수

### 3. 사용자 인터페이스
- 영상 목록 브라우징 (그리드/리스트 뷰)
- 썸네일과 요약 정보 미리보기
- 영상 시청 페이지 (YouTube 임베드)
- 검색 및 필터링 기능
- 즐겨찾기 및 시청 기록 관리

### 4. 대시보드 및 분석
- 수집된 영상 통계 (총 개수, 일별 수집량)
- 인기 채널 순위
- 조회수 기반 인기 영상
- 최근 업로드 영상 트렌드
- 주제별 영상 분포도
- 월별/주별 영상 수집 현황

### 5. 관리자 기능
- 영상 승인/거부 시스템
- 수동 영상 추가
- 수집 키워드 관리
- 시스템 로그 모니터링

## 기술 요구사항

### 프론트엔드
- Next.js 15 (React 19)
- TypeScript
- Tailwind CSS
- 반응형 디자인
- PWA 지원

### 백엔드
- Next.js API Routes 또는 Express.js
- 데이터베이스: PostgreSQL 또는 MongoDB
- Redis (캐싱 및 작업 큐)
- YouTube Data API v3
- Claude API (영상 요약 생성)

### 인프라
- Docker 컨테이너화
- 일일 데이터 수집을 위한 Cron Job
- 클라우드 배포 (Vercel/AWS/GCP)
- 이미지 CDN

## 성능 요구사항
- 페이지 로딩 시간: 3초 이내
- 동시 사용자: 100명 이상 지원
- 일일 API 호출 제한 관리
- 데이터베이스 쿼리 최적화

## 보안 요구사항
- API 키 보안 관리
- 사용자 인증/권한 관리
- CORS 정책 설정
- SQL Injection 방지
- XSS 공격 방지

## 사용자 스토리

### 일반 사용자
1. 사용자는 클로드 코드 관련 최신 YouTube 영상을 한 곳에서 확인하고 싶다
2. 사용자는 영상의 요약된 내용을 보고 시청할지 결정하고 싶다
3. 사용자는 인기 있는 영상이나 특정 채널의 영상을 쉽게 찾고 싶다
4. 사용자는 관심 있는 영상을 즐겨찾기에 추가하고 나중에 볼 수 있게 하고 싶다

### 관리자
1. 관리자는 자동 수집된 영상의 품질을 관리하고 싶다
2. 관리자는 시스템 상태와 수집 현황을 모니터링하고 싶다
3. 관리자는 검색 키워드를 조정하여 수집 정확도를 향상시키고 싶다

## 마일스톤

- 간단한 영상 목록 페이지

### Phase 2
- AI 요약 기능 추가
- 고급 필터링 및 검색
- 사용자 계정 시스템

### Phase 3

- 알림 시스템
